## ✨ Overview

**Postix** is a full-featured **Social Media Web Application** designed to deliver a smooth and interactive user experience.  
Built with **Angular 20**, it incorporates modern frontend practices, reactive state management, and scalable architecture.

Live Demo: [Postix App](https://eng-xmoamenmohamedx.github.io/Postix-social-media-app/#/)

---

## 👤 User System Features

- 🔐 Signup & Login with authentication guards  
- 🔄 Password change functionality  
- 🖼️ Upload profile & cover photos (with image cropping & zoom)  
- ✏️ Edit & manage personal profile  
- 🌍 Browse other users' profiles  
- 🤝 Follow / Unfollow users  
- 👥 Friends list & suggestions  
- 🔍 Live search and filtering  

---

## 📝 Posts System

- 🆕 Create posts with images and privacy settings  
- ✏️ Edit and delete posts inline  
- ♾️ Infinite scrolling (Intersection Observer)  
- 🔀 Multiple feed modes: Following, My Posts, Community  
- 📌 Save & unsave posts  
- 🔗 Share posts & copy link  
- ❤️ Like / Unlike posts  
- 👀 View likes & interactions  

---

## 💬 Comments & Replies

- 🧱 Full CRUD operations  
- 👍 Like / Unlike comments  
- 🧵 Nested replies system for better discussions  

---

## 🔔 Notifications System

- 🔴 Real-time unread count badge  
- 📂 Filter notifications (All / Unread)  
- ✅ Mark single or all notifications as read  

---

## 🖼️ UI & UX Enhancements

- 🔍 Full-screen image viewer  
- ⏱️ Custom TimeAgo pipe with auto-updates via RxJS  
- 💀 Skeleton loading screens  
- 🔔 Toast notifications for feedback  
- ⚠️ Confirmation modals for critical actions  
- 🔄 Scroll position restoration  
- 🚫 Custom 404 page
